# EU AI Act Compliance — Career Twin Workspace v1

## Classification Assessment

Career Twin Workspace v1 is designed to avoid HIGH-RISK classification under EU AI Act.

## v1 Hard Boundaries (prevents high-risk classification)

| Use Case | v1 Status | Reason |
|----------|----------|--------|
| HR performance evaluation | FORBIDDEN | Would constitute high-risk AI in employment |
| Promotion/demotion decisions | FORBIDDEN | Employment decision support = high-risk |
| Behavioral monitoring | FORBIDDEN | Psychological profiling at workplace |
| Biometric data collection | FORBIDDEN | Explicitly prohibited in EU AI Act |
| Automated scoring for employment | FORBIDDEN | High-risk category |

## What v1 IS (not high-risk)
- Personal skill development tool (user-controlled)
- Co-work methodology platform
- Learning progress tracker (user-owned data)
- Task quality feedback system (advisory, not decision)

## Data Handling Compliance
- User data: stored locally in workspace, not transmitted to third parties
- No persistent biometric or behavioral profiling
- User can delete all personal data (Layer A) on request
- Aggregate analytics (Layer B) are anonymized and aggregated (N>=5)

## Human Oversight
- All important outputs require user review and approval
- Twin has advisory role only — no autonomous employment decisions
- Human-in-the-loop at all 3 checkpoints per task
- Session data controlled by user, not employer

## Version 2+ Considerations
If Career Twin data is requested for HR evaluation in Phase 2 (enterprise):
- Requires explicit legal review of EU AI Act Article 6 high-risk provisions
- Requires separate data consent framework
- Requires human oversight documentation
- Requires worker notification obligations
