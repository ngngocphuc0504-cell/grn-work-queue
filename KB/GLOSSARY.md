# GLOSSARY.md — Career Twin Workspace

| Term | Definition |
|------|-----------|
| **CPT** | Core Professional Twin — bộ kỹ năng nền tảng, IMMUTABLE |
| **CST** | Current Self Twin — snapshot năng lực thực tế của user, READ-WRITE |
| **FRT** | Future Role Twin — model năng lực mục tiêu, SEMI-IMMUTABLE |
| **Co-Work Mode** | Mức độ tự chủ của twin: OBSERVE → COWORK → DELEGATED → SWARM |
| **Skill Signal** | Dữ liệu đo lường năng lực được trích từ task artifacts và reflection |
| **Skill Maturity Index** | Điểm tổng hợp 5 chiều kỹ năng (1-5 per skill) |
| **DoD** | Definition of Done — tiêu chí hoàn thành của một task |
| **Human First Rule** | User phải nêu approach trước khi twin respond — luôn luôn |
| **Framed Brief** | Task description đã qua 5-bước framing protocol (SKH-01) |
| **KI** | Knowledge Item — đơn vị tri thức được distill từ reflection |
| **Reflection** | 3-câu tổng kết sau mỗi task: worked / failed / next change |
| **Role Pack** | Domain-specific skill overlay cho FRT (BA/DA/PM/Manager/Specialist) |
| **GAP Map** | Delta visualization giữa CST và FRT |
| **TTFV** | Time-to-First-Value — thời gian từ onboarding đến lúc user thấy giá trị |
| **Skill Signal Engine** | Hệ thống thu thập, aggregate và update Skill Maturity Index |
| **Genesis Header** | Metadata block bắt buộc trên mọi task artifact |
| **BLUF** | Bottom Line Up Front — nguyên tắc viết communication (SKH-04) |
| **Executor-Swarm** | Sub-agent layer xử lý bounded tasks trong DELEGATED/SWARM mode |
