# Role Pack Library v1

## Available Role Packs (v1)

### RP-01: Business Analyst
**Competency focus:** Requirements elicitation, process mapping, stakeholder management, data-informed decision making  
**Signature outputs:** BRD, process flow, requirements doc, user story, gap analysis  
**Key skill weights:** SKH-01 (30%), SKH-02 (30%), SKH-04 (20%), SKH-03 (10%), SKH-05 (10%)  
**FRT capability targets:** BABOK core competencies, structured problem framing, evidence-based recommendations  

### RP-02: Data Analyst
**Competency focus:** Data interpretation, insight communication, pattern recognition, analytical storytelling  
**Signature outputs:** Analysis report, dashboard narrative, insight memo, recommendation brief  
**Key skill weights:** SKH-02 (35%), SKH-04 (25%), SKH-01 (20%), SKH-05 (15%), SKH-03 (5%)  
**FRT capability targets:** Statistical reasoning, data storytelling, visualization judgment  

### RP-03: Project Manager / Coordinator
**Competency focus:** Project planning, milestone tracking, stakeholder communication, risk management  
**Signature outputs:** Project plan, status report, risk log, escalation note, meeting brief  
**Key skill weights:** SKH-03 (30%), SKH-04 (25%), SKH-01 (20%), SKH-02 (15%), SKH-05 (10%)  
**FRT capability targets:** PMBOK fundamentals, dependency management, variance reporting  

### RP-04: Manager / Team Lead
**Competency focus:** Decision making, people communication, strategic framing, delegation  
**Signature outputs:** Decision note, team update, strategy memo, feedback session structure  
**Key skill weights:** SKH-04 (30%), SKH-01 (25%), SKH-02 (20%), SKH-03 (15%), SKH-05 (10%)  
**FRT capability targets:** Strategic communication, structured decision-making, effective delegation  

### RP-05: Knowledge Worker / Specialist
**Competency focus:** Deep work execution, knowledge documentation, expertise communication  
**Signature outputs:** Technical brief, knowledge article, subject matter memo, training outline  
**Key skill weights:** SKH-02 (30%), SKH-04 (25%), SKH-05 (20%), SKH-01 (15%), SKH-03 (10%)  
**FRT capability targets:** Expert communication, knowledge structuring, peer review capability  

## Role Pack Selection in Onboarding
User selects 1 primary Role Pack. Secondary pack optional (max 1).  
FRT is initialized from Role Pack competency framework + user's stated goals.
