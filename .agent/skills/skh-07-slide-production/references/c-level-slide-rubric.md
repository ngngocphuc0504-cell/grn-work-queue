# C-Level Slide Rubric

## 1. Action Titles
- The title must make a complete statement.
- Bad: "Q3 Sales Data"
- Good: "Q3 Sales Dropped 15% Due to Churn in APAC Region"

## 2. Visual Over Text
- If describing a process -> Suggest a Flowchart.
- If comparing features -> Suggest a Matrix.
- If showing trends -> Suggest a Line/Bar Chart.
- Never use more than 6 bullet points per slide.

## 3. Speaker Notes
- Must not just read the slide.
- Should provide the "So What?"
- Include anticipated Q&A triggers.

## 4. Workflow-Based Pitch (Solopreneur Lens)
- Slide thuyết trình B2B phải tập trung vào "Hệ thống thay thế con người" (FTE-replacement).
- Thay vì chỉ trình bày facts, hãy pitch các Fractional Service / Automation Workflows giúp khách hàng tối ưu hóa hệ thống. Tiết kiệm Headcount là ngôn ngữ lõi.
