# Analytical Rubric Dimensions — Domain Reference

> Reference document for skill `sga-02`.
> Defines the scoring rubric for analytical quality assessment across 3 primary dimensions.

---

## Dimension Framework

Career Twin reviews use a 3-dimension rubric for analytical artifacts. Each dimension scores 1-5 and MUST be evidence-backed by citing specific artifact passages.

---

## D1: Evidence Quality (SKH-02 Primary)

### Definition
Measures whether claims in the artifact are supported by verifiable data, citations, or logical reasoning chains — versus unsupported assertions.

### Scoring Matrix

| Score | Level | Criteria | Example |
|-------|-------|----------|---------|
| 1 | Absent | No evidence for any claim | "Revenue tăng mạnh" — no number, no source |
| 2 | Superficial | Generic data without source | "Revenue tăng 20%" — no source attribution |
| 3 | Adequate | Data with source | "Revenue tăng 20% (ERP Q1 report)" |
| 4 | Strong | Multiple sources + cross-validation logic | "Revenue tăng 20% (ERP) align with POS data (+18%)" |
| 5 | Exemplary | Sources + causal reasoning + limitations stated | "Revenue +20% driven by new SKU launch (source: ERP + POS). Limitation: seasonal spike not isolated." |

### Common Anti-Patterns
- **Weasel Words:** "Có thể", "Khả năng cao", "Thường thì" — without quantification
- **Selective Evidence:** Only citing data that supports conclusion, ignoring contradicting data
- **Circular Reference:** Claim A proven by Claim B, which is proven by Claim A

---

## D2: Logic Structure (SKH-02 Secondary)

### Definition
Measures whether the artifact follows a clear logical flow: premise → analysis → conclusion, with explicit reasoning chains connecting each step.

### Scoring Matrix

| Score | Level | Criteria |
|-------|-------|----------|
| 1 | Absent | Stream-of-consciousness, no structure |
| 2 | List-based | Bulleted items, no logical connection between them |
| 3 | Sequential | Clear flow but missing "therefore" connectives |
| 4 | Structured | Pyramid structure with "because" and "therefore" chains |
| 5 | Robust | Minto Pyramid compliant + counter-argument addressed |

---

## D3: Action Clarity (SKH-01 Secondary)

### Definition
Measures whether the analysis leads to clear, actionable recommendations — not just observations.

### Scoring Matrix

| Score | Level | Criteria |
|-------|-------|----------|
| 1 | Absent | Analysis without any recommendation |
| 2 | Vague | "Cần cải thiện" without specifics |
| 3 | Specific | Named action with responsible party |
| 4 | Prioritized | Named actions ranked by impact/effort |
| 5 | Executable | Action + owner + deadline + success metric |

|--------------------|-------|---------------|

| 3-5 | D | "Artifact cần rework. Focus: D[X] — cụ thể: [action]." |
