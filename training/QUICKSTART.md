# QUICKSTART — Career Twin Workspace v1

## Daily Session Flow

Step 1: WF-INF-01 (Start Session)
  Twin greets you with SMI summary and active project status
  You state: "Hom nay toi dang lam [task]. Approach cua toi la [X]."

Step 2: Submit task (WF-01)
  MUST state approach first — twin will ask if missing

Step 3: Execute (WF-02)
  COWORK mode: you draft, twin is available for questions
  DELEGATED/SWARM: you write delegation brief, twin executes bounded tasks

Step 4: Review (WF-03 or WF-04)
  Self-assess first ("Toi tu cham X/5 vi...")
  Twin scores with rubric and artifact evidence

Step 5: Validate (WF-05)
  You declare task done
  Twin checks each DoD item: MET / NOT MET

Step 6: Reflect (WF-06) — MANDATORY
  Write 3 sentences: worked / failed / next change
  Session CANNOT close without this

Step 7: WF-INF-02 (End Session)
  Twin generates session summary
  Background memory consolidation queued

---

## Mode Reference

Mode       | Unlock Condition        | Twin Authority
-----------|------------------------|---------------------------
OBSERVE    | Week 1 default          | Demonstrate, advise only
COWORK     | SMI avg 2.0 + 5 tasks   | Review and challenge
DELEGATED  | SMI avg 3.0 + 15 tasks  | Execute bounded sub-tasks
SWARM      | SMI avg 4.0 + 30 tasks  | Full project coordination

---

## The 3 Rules You Must Never Break

1. State your approach BEFORE asking twin anything
2. Never close session without reflection (twin will block it)
3. You own every important output — twin assists, you decide
